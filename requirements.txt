# SAMRUDDHI PETROLEUM - DSM SHIFT MANAGEMENT SYSTEM
# Requirements for running on a new laptop
# ==========================================

## SYSTEM REQUIREMENTS
- Node.js: v24.13.0 or higher (https://nodejs.org)
- npm: 10.x or higher (comes with Node.js)
- PostgreSQL: 13.x or higher (https://www.postgresql.org/download/)
- Git: Latest version (for cloning the repo)

## DATABASE REQUIREMENTS
- PostgreSQL Server running on localhost:5432
- Default credentials:
  * Username: postgres
  * Password: root (change after setup)
  * Database name: samruddhi

## PORT REQUIREMENTS
- Frontend Dev Server: 5173 (auto-falls back to 5174)
- Backend API: 4000
- PostgreSQL: 5432

## INSTALLATION STEPS
1. Install Node.js from https://nodejs.org (v24.13.0 or higher)
2. Install PostgreSQL from https://www.postgresql.org/download/
3. Clone repository: git clone <repo-url>
4. See SETUP.md for detailed installation instructions

## ENVIRONMENT SETUP
- Copy .env.example to .env in the backend folder
- Configure DATABASE_URL, JWT_SECRET, and CORS_ORIGINS
- See .env.example for all required variables

## QUICK START
After all dependencies are installed:
  cd backend && npm install && npm run prisma:generate
  cd ../frontend && npm install
  # In backend terminal: npm run dev
  # In frontend terminal: npm run dev
  # Open http://localhost:5174

For detailed setup instructions, see SETUP.md
